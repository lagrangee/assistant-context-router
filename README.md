# Assistant Context Router MVP

## 背景
本项目承接 `projects/exploration/assistant-routing-research/` 的研究结论。

目标不是重写 OpenClaw，而是交付一个：
- project-centric
- trigger-driven
- context-selective
- extension-first

的 assistant routing policy + bridge 层 MVP。

## 上游研究入口
优先阅读：
- `../../exploration/assistant-routing-research/research/research-summary.md`
- `../../exploration/assistant-routing-research/research/implementation-brief.md`
- `../../exploration/assistant-routing-research/research/implementation-constraints.md`

- `implementation/`：实现笔记、原型、接口说明
